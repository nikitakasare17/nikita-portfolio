## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/mihir2004/portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd portfolio
   ```

## Usage

Open `index.html` in your preferred web browser to view the portfolio.
